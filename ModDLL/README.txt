# Installing
1. Make sure you have the latest MS Visual C++ Redistributable https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170
2. Place the contents of this folder under \mcc\binaries\win64\ in your MCC install directory.

# Running
1. Launch MCC with EAC disabled.
2. Run SuperhotMCCLauncher.exe after the game loads.
3. Unload the mod with F9 or pause the mod with F3.

# Notes

Currently, only single player Halo CE on the Steam version of MCC is supported.

There is currentlly a bug where you may need to set your FOV to 90 degrees for tracers to draw in the right location.
